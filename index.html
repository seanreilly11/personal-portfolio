Expense Tracker

This is an expense tracker made using React and the React Context API. It takes in income and expenses and calculates your total money available as well as total income and expenses.

You are able to add and delete income and expenses and it shows a history while differentiating between income and expense.

This also has a full-stack counterpart made with the MERN stack which persists values using MongoDB.
