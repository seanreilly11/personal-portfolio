Trade Easy

Trade Easy is a full-stack C2C website that was based off of Trade Me which allows people to sell their products as well as buy then off other people. The app is fully functional except for payment methods and in place of this virtual currency is used to replicate real money.

My responsibilities in this project were to build the backend Express server and set up all the endpoints to MongoDB. I also created an account page for the user that contained information on products they were selling, had sold, had bought and that were on their eatchlist.
